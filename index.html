What is grind?

Yes... grind is another css framework. But! Its free, lightweight, responsive, open source and modular.

Grind comes with a minimal set of stylesheets with whom you are able to build responsive websites without overhead. And that is as easy as winking. Use this framework or parts of it if you are slained by the modern frameworks functionalities.
Just download the grind package, add the css folder to your web project and link the stylesheets you want to your HTML head.

Grind is separated as followed:

  1. Normalize.css -
  2. Grid - responsive, 12 columns, centerd and full width container
  3. Buttons - default, ghost
  4. Typo - ul, headings
  5. Helper Classes - floating
  6. Forms
  7. Code - clean code presentation
  8. Animations - hover effects and animations based on CSS

You can choose the readable .css files or the minified ones.

How To Use

As previously mentioned, to work with grind just link the stylesheets and add the viewport meta information. The code below is representing a clean grind html file with all modules included. 


    <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="utf-8">
    <title>grind - responsive css grid</title>
    <meta name="description" content="">
    <meta name="author" content="">

    <meta name="viewport" content="width=device-width, initial-scale=1">


    <link rel="stylesheet" href="css/normalize.min.css">
    <link rel="stylesheet" href="css/grind_grid.min.css">
    <link rel="stylesheet" href="css/grind_buttons.min.css">
    <link rel="stylesheet" href="css/grind_typo.min.css">
    <link rel="stylesheet" href="css/grind_helper_classes.min.css">
    <link rel="stylesheet" href="css/grind_forms.min.css">
    <link rel="stylesheet" href="css/grind_code.min.css">
    <link rel="stylesheet" href="css/grind_animations.min.css">

    </head>
    <body>

        Your page ...

    </body>
    </html>

Grid

The responsive grind grid is the most important module.
As followed a list of the available css classes:

  1. .one - .twelve: The common column elements
  2. .one-offset - .twelve-offset: A offset column element floats to the right side
